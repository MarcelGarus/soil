| number | mnemonic | arg 0 (a)      | arg 1 (b) | description                   |
| ------ | -------- | -------------- | --------- | ----------------------------- |
| 0      | exit     | status         | -         | Exits the program.            |
| 1      | print    | pointer to msg | len       | Prints the message to stdout. |
| 2      | log      | pointer to msg | len       | Logs the message to stderr.   |
